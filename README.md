# react-native-allinone-upi

UPI Payment using installed UPI Payment Apps in Mobile

## Installation

```sh
npm install react-native-allinone-upi
```

## Usage

```js
import { multiply } from "react-native-allinone-upi";

// ...

const result = await multiply(3, 7);
```
